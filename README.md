<h1>Decode Health</h1>

<p>
A futuristic AI-powered health platform designed to deliver personalized healthcare insights using intelligent systems, modern UI, and scalable backend architecture.
</p>

<h2>Project Overview</h2>
<p>Decode Health is a full-stack health intelligence platform that integrates:</p>
<ul>
  <li>Modern React-based UI</li>
  <li>Flask backend APIs</li>
  <li>Authentication system</li>
  <li>Database persistence</li>
  <li>AI/ML-ready chatbot architecture</li>
</ul>

<h2>Key Features</h2>
<ul>
  <li>User Authentication (Sign In / Log In)</li>
  <li>Health Profile System</li>
  <li>Chatbot UI for multiple health services</li>
  <li>Modular health service architecture</li>
  <li>Initial ML integration for Menstrual Health</li>
  <li>Clean, futuristic UI design system</li>
</ul>

<h2>Services Implemented</h2>
<ul>
  <li>Smart Assistance</li>
  <li>Menstrual Health</li>
  <li>Weight & Metabolism</li>
  <li>Nutrition Intelligence</li>
  <li>Fitness & Recovery</li>
  <li>Symptom Checker</li>
  <li>Mental Health</li>
</ul>

<h2>Tech Stack</h2>

<h3>Frontend</h3>
<ul>
  <li>React.js</li>
  <li>CSS (Custom Design System)</li>
  <li>Lucide Icons</li>
</ul>

<h3>Backend</h3>
<ul>
  <li>Flask</li>
  <li>Flask-CORS</li>
  <li>Flask-Bcrypt</li>
  <li>JWT Authentication</li>
</ul>

<h3>Database</h3>
<ul>
  <li>SQLite (via SQLAlchemy)</li>
</ul>

<h3>Machine Learning</h3>
<ul>
  <li>Scikit-learn</li>
  <li>Logistic Regression</li>
  <li>StandardScaler</li>
</ul>

<h2>System Architecture</h2>
<pre>
Frontend (React)
        ↓
Flask API (Backend)
        ↓
Database (SQLite)
        ↓
ML Model (Scikit-learn)
        ↓
Prediction Response → Chat UI
</pre>
